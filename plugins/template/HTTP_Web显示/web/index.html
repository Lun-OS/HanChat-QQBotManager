🚀 HTTP Web显示插件

静态文件服务已成功运行！

📋 插件信息

  • 插件名称: http_web_display
  • 版本: 1.0.0
  • 功能: 提供类似Nginx的静态文件服务

📁 支持的文件类型

🌐 HTML/CSS/JS
🖼️ PNG/JPG/GIF/SVG
🔤 WOFF/TTF/OTF
📄 JSON/XML/TXT
🎬 MP4/WEBM/MP3
📦 PDF/ZIP等

⚡ 功能特性

  • ✅ 自动识别 MIME 类型
  • ✅ 支持目录索引（自动查找 index.html）
  • ✅ 防止目录遍历攻击
  • ✅ 支持缓存控制
  • ✅ 404/500 错误页面

📝 使用说明

  1. 将静态文件放入插件目录的 web/ 子目录中
  2. 通过 /plugins/{selfid}/web/文件名 访问
  3. 访问 /plugins/{selfid}/web 自动加载 index.html
