# 指定消息撤回插件

## 功能说明

本插件允许管理员通过回复消息并@机器人+关键字的方式撤回指定消息。

## 使用方法

1. **管理员**在群中**回复**想要撤回的消息
2. 在回复的消息中**@机器人**并加上**关键字"撤回"**
3. 机器人会自动撤回被回复的消息

### 示例

```
[CQ:reply,id=123456789][CQ:at,qq=3851157533] 撤回
```

## 配置说明

在 `config.json` 文件中配置：

```json
{
    "group_list": [
        1070163250,
        131933642,
        342584820
    ],
    "admin": [
        3851157533,
        1596534228
    ],
    "Keywords": "撤回"
}
```

### 配置项说明

- `group_list`: 启用的群列表（留空则所有群都启用）
- `admin`: 管理员QQ号列表（只有这些用户可以撤回消息）
- `Keywords`: 触发撤回的关键字（默认"撤回"）

## 注意事项

1. 只有配置中的管理员才能使用此功能
2. 必须回复要撤回的消息才能生效
3. 消息必须在机器人可见范围内才能被撤回
4. 某些消息类型可能无法撤回（如系统消息）
5. **静默模式**：不会发送任何成功或失败提示消息到群里，只在日志中记录
