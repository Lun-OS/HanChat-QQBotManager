# 项目概述

## 项目简介

HanChat-QQBotManager 是一个功能强大的 QQ 机器人管理平台，专为 QQ 机器人开发者和管理者设计。它提供了完整的机器人管理、插件开发和运行时环境，让您能够轻松创建、部署和管理多个 QQ 机器人。

## 核心价值

### 1. 降低开发门槛

- **可视化编程**：通过 Blockly 图形化编程，无需编写代码即可创建插件
- **模板插件**：提供丰富的插件模板，开箱即用
- **详细文档**：完整的开发文档和示例代码

### 2. 提高管理效率

- **多账号管理**：统一管理多个 QQ 机器人账号
- **实时监控**：账号状态、消息日志实时查看
- **Web 界面**：现代化的 Web 管理界面，操作便捷

### 3. 强大的扩展能力

- **Lua 脚本**：灵活的 Lua 脚本支持，满足复杂需求
- **丰富 API**：提供消息、群组、用户、文件、网络等全方位 API
- **HTTP 接口**：插件可注册自定义 HTTP 接口，与外部系统集成

| 技术 | 版本 | 用途 |

### Lua 相关库

- **gopher-lua**: Go 语言实现的 Lua 5.1 解释器
- **自定义 Lua API**: 封装了消息、群组、用户、文件、网络、存储等接口

## 主要功能模块

### 1. 账号管理模块

- 多账号同时在线管理
- 账号状态实时监控
- 账号配置管理
- WebSocket 连接管理

### 2. 插件系统模块

- Lua 插件沙箱运行
- 插件热加载/热卸载
- 插件生命周期管理
- 插件配置管理
- 插件数据持久化

### 3. 消息处理模块

- OneBot 协议消息解析
- 消息事件分发
- 消息段处理
- 消息日志记录

### 4. Web 管理界面

- 仪表盘展示
- 账号管理界面
- 插件管理界面
- Blockly 编辑器
- WebQQ 聊天界面
- 日志查看界面

### 5. API 服务模块

- RESTful API 接口
- WebSocket 实时通信
- 插件自定义 HTTP 接口
- JWT 认证

## 应用场景

### 1. 社群管理

- 自动回复群消息
- 群成员管理（禁言、踢出等）
- 群公告定时发布
- 违禁词自动检测

- 定时推送新闻资讯
- 天气信息播报
- 生日提醒
- 活动通知

### 3. 数据收集

- 群消息统计
- 用户活跃度分析
- 关键词收集
- 反馈收集

### 4. 业务集成

- 通过 HTTP 接口与现有系统集成
- 接收外部系统指令
- 推送数据到外部系统
- 自动化工作流
