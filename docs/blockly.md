# Blockly 可视化编程

本文档介绍如何使用 Blockly 进行可视化插件开发。

## 什么是 Blockly

Blockly 是 Google 开发的可视化编程库，允许用户通过拖拽代码块的方式来编写程序，无需手写代码。

## 进入 Blockly 编辑器

1. 登录 Web 管理界面
2. 点击导航栏中的"Blockly 编辑器"
3. 选择您的机器人账号
4. 开始创建插件

## 界面介绍

### 工具箱（左侧）

工具箱包含所有可用的代码块，按类别分组：

- **事件**：消息事件、通知事件、请求事件
- **消息**：发送消息、撤回消息、消息解析
- **群组**：群组管理、成员管理
- **用户**：用户信息、好友管理
- **控制**：条件判断、循环
- **变量**：变量定义、变量赋值
- **存储**：数据存储、数据读取
- **日志**：日志输出
- **配置**：配置读取
- **网络**：HTTP 请求
- **工具**：字符串处理、编码解码

### 工作区（中间）

工作区是您拖拽和连接代码块的地方。

### 预览（右侧）

预览区域显示生成的 Lua 代码。

## 第一个 Blockly 插件

### 步骤 1：创建新项目

1. 点击"新建项目"
2. 输入插件名称，如"hello_world"
3. 填写插件描述
4. 点击"创建"

### 步骤 2：添加消息事件

1. 从左侧工具箱的"事件"分类中
2. 拖拽"当收到消息时"块到工作区

### 步骤 3：添加条件判断

1. 从"控制"分类中拖拽"如果...那么..."块
2. 从"消息"分类中拖拽"消息包含关键词"块
3. 将关键词设置为"你好"
4. 连接到"如果"条件

### 步骤 4：添加发送消息

1. 从"消息"分类中拖拽"发送群消息"块
2. 输入消息内容"你好！很高兴见到你！"
3. 连接到"那么"分支

### 步骤 5：保存和运行

1. 点击"保存"按钮
2. 点击"运行"按钮
3. 在群里发送"你好"测试插件

## 常用代码块示例

### 关键词回复

```
当收到消息时
    如果 消息包含关键词 "你好" 那么
        发送群消息 到 [群号] 内容 "你好！"
```

### 计数器

```
当收到消息时
    设置 count = 存储读取 "count" 默认 0
    设置 count = count + 1
    存储写入 "count" = count
    如果 count % 10 == 0 那么
        发送群消息 内容 "你已经发了 " + count + " 条消息！"
```

### 群管助手

```
当收到消息时
    如果 消息以 "#禁言" 开头 那么
        如果 发送者是管理员 那么
            提取目标 QQ 和时间
            设置群禁言 群号 [群号] 用户 [目标] 时长 [时间]
            发送群消息 内容 "已禁言"
```

## 代码块参考

### 事件块

| 块名称 | 说明 |
|--------|------|
| 当收到消息时 | 注册消息事件处理器 |
| 当收到通知时 | 注册通知事件处理器 |
| 当收到请求时 | 注册请求事件处理器 |

### 消息块

| 块名称 | 说明 |
|--------|------|
| 发送群消息 | 发送群消息 |
| 发送私聊消息 | 发送私聊消息 |
| 撤回消息 | 撤回指定消息 |
| 消息包含关键词 | 检查消息是否包含关键词 |
| 获取消息纯文本 | 获取去除 CQ 码的纯文本 |
| 获取群号 | 从消息事件获取群号 |

| 块名称 | 说明 |
|--------|------|

| 等待...秒 | 延迟执行 |

### 存储块

| 块名称 | 说明 |
|--------|------|
| 存储写入 | 写入数据到持久化存储 |
| 存储读取 | 从持久化存储读取数据 |
| 存储删除 | 删除存储中的数据 |

### 日志块

## 导出和导入

### 导出项目

1. 点击"导出"按钮
2. 项目将保存为 JSON 文件
3. 可以分享或备份

### 导入项目

1. 点击"导入"按钮
2. 选择之前导出的 JSON 文件
3. 项目将加载到工作区

## 查看 Lua 代码

在右侧预览区域，您可以实时查看生成的 Lua 代码。您也可以：

1. 点击"复制代码"将代码复制到剪贴板
2. 点击"下载代码"下载 Lua 文件

## 模板项目

系统内置了几个模板项目供您参考：

- **关键词回复示例**：简单的关键词自动回复
- **HTTP 获取图片并发送示例**：展示如何使用网络请求
- **所有接口调用示例**：展示各种 API 的使用方法

## 进阶技巧

### 使用变量

1. 从"变量"分类中拖拽"创建变量"块
2. 输入变量名
3. 使用"设置变量"和"获取变量"块操作变量

### 复杂逻辑

可以将多个条件判断块嵌套使用，实现复杂的逻辑。

### 组合使用

Blockly 生成的代码与手写 Lua 代码完全兼容，您可以：
1. 用 Blockly 生成基础框架
2. 导出 Lua 代码
3. 手动添加复杂逻辑
